Basic Account Holder Information API

By: Oluwatobiloba Oshodi & Lukman Gbadawiyu

DESCRIPTION

This API allows a caller to retrieve, submit, update or delete information on various account holders.

Routes URLs:

    To get all accounts: /accounts, For example; https://tobiloba-lukman.onrender.com/accounts

    To get specific account: /account/id, For example; https://tobiloba-lukman.onrender.com/account/639864db6a6ba62f3d117767
