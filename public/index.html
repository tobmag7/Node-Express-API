Basic Account Holder Information Express API

By: Oluwatobiloba Oshodi & Lukman Gbadawiyu

DESCRIPTION

This API allows a caller to retrieve, submit, update or delete information on Account Holders.

READ

To access the JSON databese, click here or send a GET request to the /accounts endpoint of the API using Postman. To read the information on all account holders.

Likewise, to access a specific account holders, send a GET request to the /account/id endpoint of the API using Postman, where id is the id number of the account holder.

CREATE

To add the information of a new account holder to the JSON database, send a POST request to the /signup endpoint of the API using Postman. The POST request must a body in a JSON format as follows:

    
      {
        "fullname": "full-name-of-account-holder",
        "email": "email-of-account-holder",
        "gender": "gender-of-account-holder",
        "address": "address-of-account-holder",
        "account_type": "type-of-account"
      }

UPDATE

To update the information of an existing account holder in the JSON database, send a PUT request to the /account/id endpoint of the API using Postman, where id is the id number of the specific account holder, provided the information of the account holder exists in the JSON database. The body of the PUT request must be in a JSON format as follows:

    
      {
        "fullname": "full-name-of-account-holder",
        "email": "email-of-account-holder",
        "gender": "gender-of-account-holder",
        "address": "address-of-account-holder",
        "account_type": "type-of-account"
      }

DELETE

To delete the information of an account holder from the JSON database, send a DELETE request to the /account/id endpoint of the API using Postman, where id is the id number of the account holder, provided the information of the account holder exists in the JSON database.
